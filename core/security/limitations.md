# IXORYN Limitations and Assumptions

## Assumptions
- User operates IXORYN in a trusted OS environment
- Input files are user-supplied, not live network streams
- Cryptographic passwords are user-generated

## Limitations
- ML steganalysis is probabilistic, not absolute
- Not certified as a legal forensic tool
- No live network monitoring
- No sandboxed malware execution
- Not resistant to OS-level compromise

## Cryptographic Scope
- Encryption strength depends on password entropy
- Key material is not recoverable without credentials

## Legal Disclaimer
IXORYN provides analytical results and does not make
authoritative legal determinations.
